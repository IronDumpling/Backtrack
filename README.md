## 分支命名规范
| 分支 | 命名 | 说明 |
| ------ | ------ | ------ |
| 主分支 | main | 主分支，所有提供给用户使用的正式版本|
| 开发主分支 | dev | 开发分支 （每个成员向这个分支发送pull request）
| 功能分支 | feature-* | 新功能分支，某个功能点正在开发阶段
| 修复分支 | bugfix-* | 修复bug的分支
